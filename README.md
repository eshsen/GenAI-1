# GenAI-1
## Краткое описание задачи
Разработать мобильное приложение с элементами ИИ, которое помогает людям поддерживать трезвость и справляться с зависимостями, заменяя вредные паттерны поведения на здоровые стратегии саморегуляции — через эмоциональный дневник, круглосуточного ИИ-наставника, анонимное сообщество и научно обоснованные упражнения.
## Подробное описание задачи
Зависимости часто возникают как способ справиться с эмоциональной болью, стрессом или одиночеством. Традиционные методы (например, группы поддержки) эффективны, но не всегда доступны в критический момент — особенно ночью или в изоляции.  

Наша задача — создать систему, которая:  

**1. Помогает осознавать и регулировать эмоции**
  - Пользователь ведёт эмоциональный дневник: 
    - описывает триггеры («ссора с начальником», «воспоминание о прошлом»), 
    - отмечает уровень тяги (1–10), 
    - выбирает стратегию замещения (дыхание, прогулка, звонок другу).
  - Система использует NLP для выявления паттернов:  
«Каждый понедельник после 18:00 у вас резко растёт тяга — возможно, связано с работой?»
  - Предлагает персонализированные упражнения на основе когнитивно-поведенческой терапии (КПТ).

**2. Обеспечивает круглосуточную поддержку через ИИ-наставника**
- ИИ-агент имитирует опытного наставника из сообщества выздоровления: 
    - задаёт наводящие вопросы («Что ты чувствуешь прямо сейчас?»), 
    - напоминает о прошлых успехах («Ты уже 22 дня трезв — это огромная сила!»), 
    - предлагает конкретные действия («Попробуй технику “5-4-3-2-1” прямо сейчас»).
- Агент обучен на анонимизированных данных из реальных групп поддержки (в симуляции — на синтетических диалогах). 
- Общение — конфиденциальное, без осуждения, в любой момент суток.  

**3. Мотивирует через визуализацию прогресса**
- Отображает цепочку трезвых дней с визуальными наградами (как в Duolingo): 
    - значки за недели, месяцы, 
    - сравнение с «средним пользователем».
- Позволяет ставить цели: «Продержаться до дня рождения сына». 
- Напоминает о последствиях срыва (в мягкой форме): «Если ты сорвёшься сегодня, потеряешь 37 дней прогресса».

**4. Связывает с анонимным сообществом и ресурсами**
- Анонимный форум: пользователи делятся опытом, поддерживают друг друга. 
- Каталог видео и упражнений: 
    - управление тревогой, 
    - работа с тягой, 
    - восстановление отношений.
- Все материалы основаны на проверенных методах: КПТ, ACT (терапия принятия и ответственности), 12 шагов.

## Условия реализации
✅ **Данные:**  
  - Использовать синтетические дневниковые записи и диалоги (можно сгенерировать на       основе открытых руководств по КПТ и 12 шагам). 
  - Не использовать реальные медицинские данные без этического одобрения. 
  - Все пользователи — анонимные, без персональной идентификации.  

✅ **Технологии:**   
- Python / JavaScript (для веб- или мобильного прототипа), 
- NLP: классификация эмоций, извлечение триггеров (spaCy, transformers), 
- LLM (через API или локально): для ИИ-наставника с «эмпатичным» промптом, 
- База знаний: структурированные упражнения по КПТ/ACT, 
- Интерфейс: Figma-прототип или простое веб-приложение (React/Streamlit).  

✅ **Обязательные компоненты:**   
- Эмоциональный дневник с анализом триггеров, 
- ИИ-чат для поддержки в кризисный момент, 
- Визуализация цепочки трезвых дней, 
- Хотя бы 3 интерактивных упражнения (например, «дыхание 4-7-8», «письмо себе», «анализ триггера»).

✅ **Ограничения:**   
- Приложение не заменяет терапию — это инструмент поддержки. 
- Никаких медицинских диагнозов или рекомендаций. 
- Все советы — общие, без персональных медицинских выводов.

## Проблема, которую решает задача
Миллионы людей борются с зависимостями в одиночку. Группы поддержки работают, но **недоступны в нужный момент**. Профессиональная помощь — дорога или перегружена. В результате люди срываются не из-за слабости, а из-за **отсутствия поддержки в критический момент**.  
Эта задача решает **проблему доступности и своевременности помощи**, предлагая:  
- **эмоциональную регуляцию вместо подавления**, 
- **круглосуточного, неосуждающего собеседника**, 
- **мотивацию через прогресс, а не страх**.

Результат — **цифровой компаньон на пути к трезвости**, который делает выздоровление менее одиноким и более устойчивым.
